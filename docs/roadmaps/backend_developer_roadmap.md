# Backend Developer Roadmap

This roadmap is organized from fundamentals to advanced topics so you can build
a strong foundation and ship real projects with confidence.

## 1) Internet & Web Fundamentals
- How the internet works: DNS, TCP/IP, HTTP/HTTPS, CDN
- Request/response lifecycle
- Basic frontend (HTML/CSS/JS) to understand frontend-backend interaction

## 2) Backend Language (pick one to go deep)
- Java / JavaScript (Node.js) / Python / Go / C# / PHP / Ruby / Rust
- OOP, data structures, basic algorithms
- Practice: build a simple CRUD API

## 3) Version Control (VCS)
- Git: commit, branch, merge, rebase
- GitHub/GitLab: pull requests, code review, issue tracking

## 4) Databases
- RDBMS: PostgreSQL/MySQL
- Schema design, normalization, indexes, foreign keys
- ORM vs raw SQL
- Migrations
- N+1 problem, query optimization

## 5) API & Security
- REST, GraphQL, gRPC
- Auth: JWT, OAuth2, refresh tokens
- CORS, HTTPS, OWASP Top 10
- Rate limiting, throttling

## 6) Caching & Performance
- Redis/Memcached
- Server-side cache, HTTP cache
- Cache invalidation, TTL, cache stampede
- Benchmarking (k6/JMeter)

## 7) Web Servers & Reverse Proxy
- Nginx / Caddy / Apache
- Basic load balancing
- TLS termination

## 8) Advanced Databases
- Transactions, ACID, isolation levels
- Failure modes, deadlocks, locking
- Replication, backup/restore

## 9) Testing
- Unit tests
- Integration tests
- E2E tests (API)
- Test coverage, mocking

## 10) Containerization & Deployment
- Docker, Docker Compose
- Environment config, secrets
- Basic cloud deployment (VM/container)

## 11) Message Brokers
- RabbitMQ, Kafka
- Async processing, retries, DLQ
- Event-driven architecture

## 12) CI/CD
- Build, test, deploy pipelines
- GitHub Actions / GitLab CI / Jenkins
- Deployment strategies: blue-green, canary

## 13) Search Engines
- Elasticsearch / Solr
- Indexing pipelines, query tuning

## 14) System Architecture
- Monolith vs microservices vs serverless
- Twelve-Factor App
- DDD, clean architecture (as needed)

## 15) Realtime & Streaming
- WebSockets, SSE
- Pub/Sub

## 16) Scaling Databases
- Sharding, replication
- CAP theorem
- Read/write splitting

## 17) NoSQL
- Document (MongoDB)
- Key-value (Redis)
- Column (Cassandra)
- Graph (Neo4j)
- Time-series (InfluxDB)

## 18) Observability & Reliability
- Logging, metrics, tracing
- Monitoring: Prometheus + Grafana
- Alerting & SLO/SLA
- Graceful degradation, circuit breaker, backpressure

## Practical Notes
- Since you are already building the ecommerce project, use it as the main
  hands-on track for each milestone instead of separate mini projects.
- Focus on one primary language before branching out.
- Keep a clear portfolio with docs and tests.
